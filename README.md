# منظم المهام والعادات

تطبيق Flutter عربي خفيف لتنظيم مهام اليوم والعادات اليومية بدون إنترنت.

## النسخة الثانية

- إضافة مهام اليوم.
- تحديد تاريخ المهمة.
- تمييز المهام التي تاريخها غدًا، اليوم، أو أصبحت متأخرة بألوان واضحة.
- تنبيه صوتي بسيط داخل التطبيق عند وجود مهام تاريخها اليوم أو متأخرة.
- إضافة عادات يومية.
- تحديد المهمة أو العادة كمنجزة.
- المهمة تعتبر منجزة مرة واحدة، أما العادة فتُحسب يوميًا.
- ملخص يومي: المهام، العادات، ونسبة الإنجاز.
- قائمة منفصلة للمهام وقائمة للعادات.
- سجل آخر 7 أيام للعادات.
- حفظ البيانات محليًا على الجهاز.
- تصميم عربي هادئ ومريح.

## طريقة البناء عبر GitHub

ارفع الملفات إلى GitHub، ثم شغل workflow باسم:

`Build APK Direct`

بعد نجاح البناء سيظهر ملف APK مباشرة داخل:

`apk/habits-tasks.apk`

وسيظهر ملف:

`DOWNLOAD_APK.md`

وفيه رابط التحميل المباشر.

## ملاحظات

- التطبيق Android فقط.
- لا يحتاج إنترنت أثناء الاستخدام.
- لا يحتاج صلاحيات حساسة.
- الحد الأدنى المقترح: Android 6 وما فوق.
- التنبيه الصوتي في هذه النسخة يعمل داخل التطبيق عند فتحه، وليس كإشعار خلفية خارج التطبيق.
